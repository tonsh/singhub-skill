# SingHub OpenClaw Skill

Search Singapore community activities and courses (OnePA + ActiveSG) from OpenClaw.

## 安装

```bash
openclaw install github:anthropic-sg/singhub-openclaw-skill
```

## 功能

- 中英文自然语言搜索
- 自动识别地区、分类、年龄、价格等筛选条件
- 返回 Telegram Mini App 入口，可直接浏览和调整筛选结果

## 示例

- "淡滨尼有什么游泳课"
- "kids swimming class near Bedok"
- "有没有免费的瑜伽"

## API

本 Skill 调用 SingHub 公开 API，无需 API key。
